# DS_Final
### 語言
`C++`

### 執行方式
先compile
Binary Tree 程式碼檔案 為 `binaryTree.cpp`
`g++ binaryTree.cpp -o binary`

Splay Tree 程式碼檔案 為 `splayTree.cpp`
`g++ splayTree.cpp -o splay`

compile之後出現執行檔案
`binary`
`splay`

直接執行後可輸出
`BTree.txt`
`BTree_PRep.txt`
`BTree_boundary.txt`
`STree.txt`
`STree_PRep.txt`
`STree_boundary.txt`
